# /prd-create — PRD для большой задачи 1С

Генерирует структурированный PRD-документ для крупной задачи (L4: 6+ объектов, сложные взаимозависимости, 100+ turns бюджета). Помогает зафиксировать ТЗ до начала кодирования и снизить scope creep.

## Параметры

| Параметр | Обязательный | Описание |
|----------|:------------:|----------|
| JsonPath | да | Путь к JSON с описанием требований |
| OutputPath | да | Путь куда писать готовый .md (обычно `docs/specs/prds/<task_id>.md`) |

## Команда

```powershell
python .claude/skills/prd-create/scripts/prd-create.py -JsonPath "tmp/req.json" -OutputPath "docs/specs/prds/inv-2026.md"
```

## JSON формат

```json
{
  "title": "Краткое название задачи",
  "context": "Зачем нужна задача, какая проблема решается",
  "scope": {
    "in":  ["Что входит в задачу"],
    "out": ["Что НЕ входит (опционально)"]
  },
  "objects": {
    "new": [
      "Catalog.НовыйСправочник (реквизиты: ...)",
      "InformationRegister.НовыйРегистр (измерения: ...)"
    ],
    "modified": [
      "Document.СуществующийДок · добавить реквизит ..."
    ],
    "removed": []
  },
  "requirements": [
    "FR-001: Юзер может ввести данные через форму X",
    "FR-002: Документ автоматически проводится по регистру Y"
  ],
  "phases": [
    "1. Создать справочники-предки",
    "2. Создать документы и регистры",
    "3. Изменения существующих документов и форм",
    "4. Финальная BSL-логика"
  ],
  "acceptance": [
    "Проверка 1: ...",
    "Проверка 2: ..."
  ]
}
```

Все поля кроме `title` опциональны. Минимально достаточно `title` + `objects.new`.

## Workflow

1. Агент получает большое ТЗ от пользователя.
2. Если задача L4 — собирает JSON-описание (через `meta_info`, `bsl_metadatasearch`, контекст диалога).
3. Записывает JSON через `Write` в `agenter/data/<project_id>/proposals/req-<task_id>.json`.
4. Вызывает `skill_run("prd-create", {json_path: ..., output_path: ...})`.
5. PRD появляется в `docs/specs/prds/<task_id>.md` — фиксированная версия ТЗ, на которую можно ссылаться.
6. Дальше — TodoWrite по фазам + кодирование.
